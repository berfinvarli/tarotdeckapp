# TarotDeck App

**TarotDeck** is a mobile application built with **React Native & Expo** that allows users to draw 3 tarot cards and read their meanings. The app is fully responsive for both mobile and tablet devices.

---

## Features

- Draw 3 tarot cards and see their meanings.
- Shuffle cards for a new reading.
- Reset button to start a new reading.
- Responsive design for both mobile and desktop.
- Bilingual meanings. (English / Turkish)
- Smooth UI and user-friendly experience.

---

## Installation

1. Clone the repo:

```bash
git clone https://github.com/YOUR_USERNAME/TarotDeck.git
cd TarotDeck
````  

2.  Install dependencies:
```bash
npm install
````

3. Start the app:
```bash
npx expo start
````
---

- Use Expo Go to run on your device (iOS or Android).


- For remote testing, enable Tunnel mode in Expo.
